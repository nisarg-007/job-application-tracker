Create Your Google Sheet

We need a Google Sheet to store your applications. Follow these steps:

1
Open Google Sheets

Go to sheets.new — this creates a blank spreadsheet instantly.

2
Name the sheet tab

At the bottom of the screen, double-click the tab name and rename it to exactly: Applications

3
Add column headers

Type these headers in Row 1, one per cell (A1 through L1):

A: Timestamp
B: Application Date
C: Job Title
D: Company Name
E: URL
F: Location
G: Salary
H: Resume Used
I: Confidence Level
J: Source
K: Status
L: Notes
💡 Pro tip: Bold the header row and freeze it (View → Freeze → 1 row) so it always stays visible.

Set Up the Backend Script

The extension sends data to a tiny script that writes to your sheet. Here's how to set it up:

1
Open Apps Script

In your Google Sheet: Extensions → Apps Script

2
Replace all code

Delete everything in Code.gs, then click the button below to copy the script code, and paste it in.

📄 Code.gs 
Loading script code...
3
Save & Deploy

Click 💾 Save, then Deploy → New deployment

4
Configure deployment

Click ⚙ gear → Web app

  • Execute as: Me
  • Who has access: Anyone
5
Authorize & Copy URL

Click Deploy → Authorize → Allow → Copy the Web App URL

🛡️ Seeing "Google hasn't verified this app"? That's normal!
This warning appears for all personal scripts. It's YOUR script accessing YOUR sheet — it's completely safe.

To proceed: Click "Advanced" (bottom-left) → Click "Go to [your project name] (unsafe)" → Click "Allow".
⚠️ The URL looks like: https://script.google.com/macros/s/AKfycb.../exec
Keep this page open — you'll paste the URL in the next step!
